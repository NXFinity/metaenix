Frontend Documentation

Welcome to the Meta EN|IX Frontend Documentation. This documentation covers the frontend web application built with React, Next.js, and modern UI frameworks.

Overview​

The Meta EN|IX frontend is a modern, responsive web application that provides a user-friendly interface for the Meta EN|IX social platform.

Technology Stack​

  • React - UI library for building user interfaces
  • Next.js - React framework for production
  • TypeScript - Type-safe JavaScript
  • Tailwind CSS - Utility-first CSS framework
  • Shadcn/UI - High-quality React components

Getting Started​

This documentation will help you understand:

  • Component architecture
  • Routing and navigation
  • State management
  • API integration
  • Styling and theming
  • Best practices
