Getting Started

Welcome to the Meta EN|IX Developer Documentation. This guide will help you integrate your application with the Meta EN|IX API to access user data, authentication, and platform services.

Overview​

Meta EN|IX provides a production-ready REST API for building applications that integrate with our platform. Whether you're building a mobile app, web application, or third-party service, our API gives you secure access to user management, authentication, content, and storage services.

Base URL​

All API requests should be made to:

https://api.metaenix.com/v1

Quick Start Guide​

To get started building with the Meta EN|IX API, follow these steps:

  1. App Setup - Register your application and obtain your credentials
  2. OAuth Authentication - Learn how to authenticate users and obtain access tokens
  3. Scopes - Understand available permissions and how to request them

What's Next?​

Once you've set up your application and understand OAuth, explore our API documentation:

  • Users API - User profiles, privacy settings, and account management
  • Posts API - Create and manage posts, comments, likes, and shares
  • Storage API - File upload and management

Key Features​

  • ✅ OAuth 2.0 - Industry-standard authentication and authorization
  • ✅ Two-Factor Authentication (2FA) - Enhanced security with TOTP support
  • ✅ Role-based access control (RBAC) - Fine-grained permissions
  • ✅ WebSocket support - Real-time communication
  • ✅ File storage - Digital Ocean Spaces integration
  • ✅ Rate limiting - Fair usage policies
  • ✅ Comprehensive documentation - Complete API reference with examples
