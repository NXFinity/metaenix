Best Practices

Follow these best practices to build reliable, secure, and efficient integrations with the Meta EN|IX API.

Security​

1. Protect Your Credentials​

Never expose client secrets or websocketIds in client-side code.

// ❌ BAD - Exposing credentials
const clientSecret = 'your-secret'; // In frontend code

// ✅ GOOD - Keep secrets server-side
// Store in environment variables
const clientSecret = process.env.CLIENT_SECRET;

2. Use HTTPS Always​

Always use HTTPS in production:

// ❌ BAD
const apiUrl = 'http://api.metaenix.com/v1';

// ✅ GOOD
const apiUrl = 'https://api.metaenix.com/v1';

3. Validate Redirect URIs​

Always validate redirect URIs server-side:

function validateRedirectUri(uri, allowedUris) {
  // Must match exactly (including protocol and trailing slashes)
  return allowedUris.some(allowed => {
    try {
      const allowedUrl = new URL(allowed);
      const uriUrl = new URL(uri);
      return allowedUrl.origin === uriUrl.origin && 
             allowedUrl.pathname === uriUrl.pathname;
    } catch {
      return false;
    }
  });
}

4. Store Tokens Securely​

// ✅ GOOD - Secure token storage
// Server-side: Use encrypted storage
const encryptedToken = encrypt(accessToken);

// Client-side: Use httpOnly cookies or secure storage
// Never store in localStorage for sensitive apps

5. Implement Token Refresh​

class TokenManager {
  constructor() {
    this.accessToken = null;
    this.refreshToken = null;
    this.expiresAt = null;
  }
  
  async getValidToken() {
    // Check if token is expired or about to expire
    if (!this.accessToken || this.isExpired()) {
      await this.refresh();
    }
    return this.accessToken;
  }
  
  isExpired() {
    if (!this.expiresAt) return true;
    // Refresh if expires within 5 minutes
    return Date.now() >= (this.expiresAt - 5 * 60 * 1000);
  }
  
  async refresh() {
    const response = await fetch('https://api.metaenix.com/v1/oauth/token', {
      method: 'POST',
      headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
      body: new URLSearchParams({
        grant_type: 'refresh_token',
        refresh_token: this.refreshToken,
        client_id: CLIENT_ID,
        client_secret: CLIENT_SECRET
      })
    });
    
    const data = await response.json();
    this.accessToken = data.access_token;
    this.refreshToken = data.refresh_token;
    this.expiresAt = Date.now() + (data.expires_in * 1000);
  }
}

Performance​

1. Implement Caching​

Cache responses when appropriate:

class ApiCache {
  constructor(ttl = 60000) {
    this.cache = new Map();
    this.ttl = ttl;
  }
  
  async get(key, fetchFn) {
    const cached = this.cache.get(key);
    
    if (cached && Date.now() - cached.timestamp < this.ttl) {
      return cached.data;
    }
    
    const data = await fetchFn();
    this.cache.set(key, {
      data,
      timestamp: Date.now()
    });
    
    return data;
  }
  
  invalidate(pattern) {
    for (const key of this.cache.keys()) {
      if (key.includes(pattern)) {
        this.cache.delete(key);
      }
    }
  }
}

2. Use Pagination Efficiently​

async function fetchAllPages(endpoint, options = {}) {
  const results = [];
  let page = 1;
  let hasMore = true;
  
  while (hasMore) {
    const response = await fetch(`${endpoint}?page=${page}&limit=100`, options);
    const data = await response.json();
    
    results.push(...data.data);
    
    hasMore = page < data.meta.totalPages;
    page++;
    
    // Rate limit consideration
    if (hasMore) {
      await new Promise(resolve => setTimeout(resolve, 100));
    }
  }
  
  return results;
}

3. Batch Operations​

Group related operations when possible:

// ❌ BAD - Multiple individual requests
for (const postId of postIds) {
  await fetch(`/v1/posts/${postId}/like`, { method: 'POST' });
}

// ✅ GOOD - Batch request if available
await fetch('/v1/posts/batch-like', {
  method: 'POST',
  body: JSON.stringify({ postIds })
});

4. Implement Request Deduplication​

class RequestDeduplicator {
  constructor() {
    this.pending = new Map();
  }
  
  async request(key, requestFn) {
    if (this.pending.has(key)) {
      return this.pending.get(key);
    }
    
    const promise = requestFn().finally(() => {
      this.pending.delete(key);
    });
    
    this.pending.set(key, promise);
    return promise;
  }
}

Error Handling​

1. Implement Retry Logic​

async function fetchWithRetry(url, options, maxRetries = 3) {
  for (let attempt = 0; attempt < maxRetries; attempt++) {
    try {
      const response = await fetch(url, options);
      
      // Don't retry client errors (4xx)
      if (response.status >= 400 && response.status < 500) {
        return response;
      }
      
      // Retry server errors (5xx) and network errors
      if (response.ok || attempt === maxRetries - 1) {
        return response;
      }
      
      // Exponential backoff
      await new Promise(resolve => 
        setTimeout(resolve, Math.pow(2, attempt) * 1000)
      );
    } catch (error) {
      if (attempt === maxRetries - 1) throw error;
      await new Promise(resolve => 
        setTimeout(resolve, Math.pow(2, attempt) * 1000)
      );
    }
  }
}

2. Handle Rate Limits Gracefully​

async function fetchWithRateLimitHandling(url, options) {
  const response = await fetch(url, options);
  
  if (response.status === 429) {
    const error = await response.json();
    const resetTime = new Date(error.resetAt);
    const waitTime = Math.max(0, resetTime.getTime() - Date.now());
    
    if (waitTime > 0) {
      await new Promise(resolve => setTimeout(resolve, waitTime));
      return fetch(url, options); // Retry
    }
  }
  
  return response;
}

3. Validate Responses​

function validateResponse(response, expectedStatus = 200) {
  if (response.status !== expectedStatus) {
    throw new Error(`Unexpected status: ${response.status}`);
  }
  
  const contentType = response.headers.get('content-type');
  if (!contentType || !contentType.includes('application/json')) {
    throw new Error('Expected JSON response');
  }
  
  return true;
}

Code Organization​

1. Create an API Client Class​

class MetaEnixClient {
  constructor(clientId, clientSecret, baseUrl = 'https://api.metaenix.com/v1') {
    this.clientId = clientId;
    this.clientSecret = clientSecret;
    this.baseUrl = baseUrl;
    this.tokenManager = new TokenManager();
  }
  
  async request(method, path, options = {}) {
    const token = await this.tokenManager.getValidToken();
    
    const response = await fetch(`${this.baseUrl}${path}`, {
      method,
      headers: {
        'Authorization': `Bearer ${token}`,
        'Content-Type': 'application/json',
        ...options.headers
      },
      ...options
    });
    
    if (!response.ok) {
      const error = await response.json();
      throw new ApiError(error);
    }
    
    return response.json();
  }
  
  async get(path, options) {
    return this.request('GET', path, options);
  }
  
  async post(path, body, options) {
    return this.request('POST', path, {
      ...options,
      body: JSON.stringify(body)
    });
  }
  
  // Add more methods as needed
}

2. Use TypeScript for Type Safety​

interface User {
  id: string;
  username: string;
  email: string;
  displayName: string;
}

interface ApiResponse<T> {
  data: T;
  meta?: {
    page: number;
    limit: number;
    total: number;
    totalPages: number;
  };
}

class MetaEnixClient {
  async getUser(id: string): Promise<User> {
    return this.get(`/users/${id}`);
  }
  
  async getUsers(page = 1): Promise<ApiResponse<User[]>> {
    return this.get(`/users?page=${page}`);
  }
}

Monitoring​

1. Log API Calls​

class LoggingClient extends MetaEnixClient {
  async request(method, path, options) {
    const startTime = Date.now();
    
    try {
      const response = await super.request(method, path, options);
      const duration = Date.now() - startTime;
      
      console.log(`API Call: ${method} ${path} - ${duration}ms`);
      return response;
    } catch (error) {
      const duration = Date.now() - startTime;
      console.error(`API Error: ${method} ${path} - ${duration}ms`, error);
      throw error;
    }
  }
}

2. Monitor Rate Limits​

class RateLimitMonitor {
  constructor() {
    this.stats = new Map();
  }
  
  recordRequest(endpoint, response) {
    const remaining = parseInt(
      response.headers.get('X-RateLimit-Remaining') || '0'
    );
    const limit = parseInt(
      response.headers.get('X-RateLimit-Limit') || '0'
    );
    
    if (!this.stats.has(endpoint)) {
      this.stats.set(endpoint, { requests: 0, rateLimited: 0 });
    }
    
    const stats = this.stats.get(endpoint);
    stats.requests++;
    
    if (remaining === 0) {
      stats.rateLimited++;
      console.warn(`Rate limit reached for ${endpoint}`);
    }
  }
}

Summary​

  • ✅ Security: Protect credentials, use HTTPS, validate inputs
  • ✅ Performance: Cache responses, use pagination efficiently, batch operations
  • ✅ Error Handling: Implement retry logic, handle rate limits, validate responses
  • ✅ Code Organization: Create API client classes, use TypeScript
  • ✅ Testing: Mock API responses, test error scenarios
  • ✅ Monitoring: Log API calls, monitor rate limits
