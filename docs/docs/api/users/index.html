Users API

The Users API allows you to manage user profiles, privacy settings, and account information.

Base Endpoint​

GET|PATCH|DELETE /v1/users/*

Get Current User​

Get the authenticated user's profile information.

Endpoint​

GET /v1/users/me

Headers​

Authorization: Bearer YOUR_ACCESS_TOKEN

Example Request​

const response = await fetch('https://api.metaenix.com/v1/users/me', {
  headers: {
    'Authorization': `Bearer ${accessToken}`,
  },
});

const user = await response.json();

Response​

{
  "id": "123e4567-e89b-12d3-a456-426614174000",
  "username": "johndoe",
  "displayName": "John Doe",
  "email": "john@example.com",
  "websocketId": "123e4567-e89b-12d3-a456-426614174001",
  "role": "Member",
  "isPublic": true,
  "profile": {
    "firstName": "John",
    "lastName": "Doe",
    "bio": "Software developer passionate about building great products",
    "avatar": "https://spaces.example.com/avatar.jpg",
    "location": "San Francisco, CA",
    "website": "https://johndoe.com",
    "dateOfBirth": "1990-01-15"
  },
  "privacy": {
    "profileVisibility": "public",

Get User by ID​

Get a specific user's public profile.

Endpoint​

GET /v1/users/:id

Parameters​

ParameterTypeDescription
idstringUser UUID

Example Request​

const userId = '123e4567-e89b-12d3-a456-426614174000';

const response = await fetch(`https://api.metaenix.com/v1/users/${userId}`, {
  headers: {
    'Authorization': `Bearer ${accessToken}`,
  },
});

const user = await response.json();

Response​

{
  "id": "123e4567-e89b-12d3-a456-426614174000",
  "username": "johndoe",
  "displayName": "John Doe",
  "isPublic": true,
  "profile": {
    "firstName": "John",
    "lastName": "Doe",
    "bio": "Software developer passionate about building great products",
    "avatar": "https://spaces.example.com/avatar.jpg",
    "location": "San Francisco, CA",
    "website": "https://johndoe.com"
  },
  "dateCreated": "2025-01-15T10:00:00Z"
}

Note: Private information (email, date of birth) is only included if you're viewing your own profile or have appropriate permissions.

Update Profile​

Update the authenticated user's profile.

Endpoint​

PATCH /v1/users/me

Headers​

Authorization: Bearer YOUR_ACCESS_TOKEN
Content-Type: application/json

Request Body​

{
  "displayName": "John Doe Updated",
  "profile": {
    "firstName": "John",
    "lastName": "Doe",
    "bio": "Updated bio text",
    "location": "New York, NY",
    "website": "https://newwebsite.com"
  }
}

Example Request​

const response = await fetch('https://api.metaenix.com/v1/users/me', {
  method: 'PATCH',
  headers: {
    'Authorization': `Bearer ${accessToken}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    displayName: 'John Doe Updated',
    profile: {
      firstName: 'John',
      lastName: 'Doe',
      bio: 'Updated bio text',
      location: 'New York, NY',
      website: 'https://newwebsite.com',
    },
  }),
});

const updatedUser = await response.json();

Response​

{
  "id": "123e4567-e89b-12d3-a456-426614174000",
  "username": "johndoe",
  "displayName": "John Doe Updated",
  "profile": {
    "firstName": "John",
    "lastName": "Doe",
    "bio": "Updated bio text",
    "location": "New York, NY",
    "website": "https://newwebsite.com"
  },
  "dateUpdated": "2025-01-20T16:00:00Z"
}

Update Privacy Settings​

Update user privacy preferences.

Endpoint​

PATCH /v1/users/me

Request Body​

{
  "isPublic": false,
  "privacy": {
    "profileVisibility": "private",
    "emailVisibility": "private",
    "showActivityStatus": false
  }
}

Example Request​

const response = await fetch('https://api.metaenix.com/v1/users/me', {
  method: 'PATCH',
  headers: {
    'Authorization': `Bearer ${accessToken}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    isPublic: false,
    privacy: {
      profileVisibility: 'private',
      emailVisibility: 'private',
      showActivityStatus: false,
    },
  }),
});

const updatedUser = await response.json();

Upload Avatar​

Upload a profile avatar image.

Endpoint​

POST /v1/users/me/avatar

Headers​

Authorization: Bearer YOUR_ACCESS_TOKEN
Content-Type: multipart/form-data

Request Body​

Form data with avatar file field.

Example Request​

const formData = new FormData();
formData.append('avatar', avatarFile);

const response = await fetch('https://api.metaenix.com/v1/users/me/avatar', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${accessToken}`,
  },
  body: formData,
});

const result = await response.json();

Response​

{
  "avatar": "https://spaces.example.com/avatars/user-123.jpg",
  "message": "Avatar updated successfully"
}

Delete Account​

Permanently delete the authenticated user's account.

Endpoint​

DELETE /v1/users/me

Headers​

Authorization: Bearer YOUR_ACCESS_TOKEN

Example Request​

const response = await fetch('https://api.metaenix.com/v1/users/me', {
  method: 'DELETE',
  headers: {
    'Authorization': `Bearer ${accessToken}`,
  },
});

const result = await response.json();

Response​

{
  "message": "Account deleted successfully"
}

Warning: This action cannot be undone. All user data, posts, and files will be permanently deleted.

Error Responses​

401 Unauthorized​

{
  "statusCode": 401,
  "message": "Unauthorized"
}

403 Forbidden​

{
  "statusCode": 403,
  "message": "You don't have permission to access this resource"
}

404 Not Found​

{
  "statusCode": 404,
  "message": "User not found"
}

Required Scopes​

  • read:user - Read user profile information
  • write:user - Update user profile and settings
  • read:user:email - Read user's email address (for email field)
