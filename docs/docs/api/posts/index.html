Posts API

The Posts API allows you to create, read, update, and delete posts, as well as interact with them through comments, likes, shares, and bookmarks.

Base Endpoint​

GET|POST|PATCH|DELETE /v1/posts/*

Create Post​

Create a new text post.

Endpoint​

POST /v1/posts

Headers​

Authorization: Bearer YOUR_ACCESS_TOKEN
Content-Type: application/json

Request Body​

{
  "content": "This is my post content",
  "isPublic": true,
  "allowComments": true
}

Example Request​

const response = await fetch('https://api.metaenix.com/v1/posts', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${accessToken}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    content: 'This is my post content',
    isPublic: true,
    allowComments: true,
  }),
});

const post = await response.json();

Response​

{
  "id": "post-uuid-123",
  "content": "This is my post content",
  "author": {
    "id": "user-uuid",
    "username": "johndoe",
    "displayName": "John Doe",
    "avatar": "https://spaces.example.com/avatar.jpg"
  },
  "isPublic": true,
  "allowComments": true,
  "likesCount": 0,
  "commentsCount": 0,
  "sharesCount": 0,
  "createdAt": "2025-01-15T10:00:00Z",
  "updatedAt": "2025-01-15T10:00:00Z"
}

Create Post with Files​

Create a post with uploaded media files.

Endpoint​

POST /v1/posts/upload

Headers​

Authorization: Bearer YOUR_ACCESS_TOKEN
Content-Type: multipart/form-data

Request Body​

Form data with:

  • content (string) - Post content text
  • files (file[]) - Media files (images, videos, GIFs) - up to 10 files, 100MB each
  • documents (file[]) - Document files (PDF, Word, Excel) - up to 5 files, 50MB each
  • isPublic (boolean) - Whether post is public
  • allowComments (boolean) - Whether comments are allowed

Example Request​

const formData = new FormData();
formData.append('content', 'Check out these images!');
formData.append('files', imageFile1);
formData.append('files', imageFile2);
formData.append('isPublic', 'true');
formData.append('allowComments', 'true');

const response = await fetch('https://api.metaenix.com/v1/posts/upload', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${accessToken}`,
  },
  body: formData,
});

const post = await response.json();

Response​

{
  "id": "post-uuid-123",
  "content": "Check out these images!",
  "media": [
    {
      "fileKey": "file-key-1",
      "url": "https://spaces.example.com/file-key-1.jpg",
      "type": "image",
      "mimeType": "image/jpeg"
    },
    {
      "fileKey": "file-key-2",
      "url": "https://spaces.example.com/file-key-2.jpg",
      "type": "image",
      "mimeType": "image/png"
    }
  ],
  "author": {
    "id": "user-uuid",
    "username": "johndoe",
    "displayName": "John Doe"
  },
  "isPublic": true,
  "allowComments": true,
  "likesCount": 0,
  "commentsCount": 0,
  "sharesCount": 0,
  "createdAt": "2025-01-15T10:00:00Z"

Parameters​

Example Request​

const post = await response.json();

Get Posts Feed​

Get a paginated list of posts.

Endpoint​

GET /v1/posts

Query Parameters​

ParameterTypeDescription
pagenumberPage number (default: 1)
limitnumberItems per page (default: 20, max: 100)
userIdstringFilter by user ID (optional)

Example Request​

const response = await fetch('https://api.metaenix.com/v1/posts?page=1&limit=20', {
  headers: {
    'Authorization': `Bearer ${accessToken}`,
  },
});

const data = await response.json();

Response​

{
  "data": [
    {
      "id": "post-uuid-1",
      "content": "First post",
      "author": { "id": "user-1", "username": "user1" },
      "likesCount": 10,
      "commentsCount": 5,
      "createdAt": "2025-01-15T10:00:00Z"
    },
    {
      "id": "post-uuid-2",
      "content": "Second post",
      "author": { "id": "user-2", "username": "user2" },
      "likesCount": 8,
      "commentsCount": 3,
      "createdAt": "2025-01-15T09:00:00Z"
    }
  ],
  "meta": {
    "page": 1,
    "limit": 20,
    "total": 50,
    "totalPages": 3
  }
}

Update your own post.

Endpoint​

PATCH /v1/posts/:id

Request Body​

{
  "content": "Updated post content",
  "isPublic": false,
  "allowComments": false
}

Example Request​

const response = await fetch(`https://api.metaenix.com/v1/posts/${postId}`, {
  method: 'PATCH',
  headers: {
    'Authorization': `Bearer ${accessToken}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    content: 'Updated post content',
    isPublic: false,
    allowComments: false,
  }),
});

const updatedPost = await response.json();

Create a collection to organize posts.

Request Body​

const response = await fetch('https://api.metaenix.com/v1/posts/collections', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${accessToken}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    name: 'My Favorites',
    description: 'Posts I want to save',
  }),
});

const collection = await response.json();

Required Scopes​

  • read:post - Read posts and comments
  • write:post - Create, update, and delete posts
  • write:comment - Create and manage comments
  • write:reaction - Like, share, and bookmark posts

Rate Limits​

  • Create Post: 10 posts per minute
  • Like/Comment/Share: 30 actions per minute
