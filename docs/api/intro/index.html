API Documentation

Welcome to the Meta EN|IX API Documentation. This documentation provides comprehensive details about all services, endpoints, request/response formats, and internal API structure.

Overview​

This documentation is intended for:

  • Internal developers working on the Meta EN|IX platform
  • System administrators managing API services
  • Contributors to the Meta EN|IX codebase
  • Anyone needing complete technical details about the API architecture

Base URL​

All API requests should be made to:

https://api.metaenix.com/v1

Authentication​

The API uses OAuth 2.0 for authentication.

Response Format​

All API responses follow a consistent JSON format.

Rate Limits​

API requests are subject to rate limiting.

Support​

For questions about the API:

  • Discord: https://discord.gg/ThCJ6mbaH8
  • Project Website: https://metaenix.com
